This project involves detecting and tracking human poses using computer vision techniques. It leverages machine learning models to identify key points on the human body and track movements in real-time. Technologies used include TensorFlow, OpenCV, and JavaScript.

Mall Clustering Project

This project focuses on segmenting customers of a mall into different clusters based on their purchasing behavior. It uses clustering algorithms like K-means to identify distinct customer groups. Technologies used include Python, Pandas, scikit-learn, and Matplotlib for visualization. The project aims to help the mall management understand customer segments and tailor marketing strategies accordingly.
